## Warehouse Storage Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/7752f4b05016430b95190b39ae2d4f59/ghb20250919) 


### 1. Related Technologies
Backend: Java, JavaWeb / Springboot/Springmvc/
Frontend: Vue3, HTML / CSS / Javascript, etc.
Database: MySQL8.0/mybatis

### 2. Related Software (Any one of the listed software can be used)
- IDEA
- Eclipse
- Visual Studio Code(VScode)
- Navicat
- etc.

#### Frontend Environment: node16
#### Java: IDEA recommended

### 3. Function Description
Homepage ECharts for retrieving backend warehouse data
Personal information settings
Warehouse management
Goods management
Employee management
Product search management
Personal information editing
Secure logout
Fullscreen mode
Day/night toggle mode
Built-in music player function
